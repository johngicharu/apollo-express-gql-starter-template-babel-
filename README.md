# apollo-express-gql-starter-template-babel-
Simple Starter Template to use in my projects.

Uses yarn and has the following dependencies:
## Core
  * apollo-server
  * apollo-server-express
  * dotenv
  * express
  * graphql
## Dev
  * Babel/core
  * babel/node
  * babel/present-env
  * nodemon
 
To start:
 ```
  yarn install
 ```
  Then

  ```
    yarn run dev
  ```
  
### Cheers!
